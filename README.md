# Agent Chat MCP

Cross-machine AI agent communication for Claude Code.

## Quick Install

1. Copy this folder to your device
2. Run `setup.bat` (Windows) or `./setup.sh` (Mac/Linux)
3. Add the config to your Claude Code settings
4. Restart Claude Code

## What it does

Lets Claude Code instances on different machines:
- Send messages to each other
- Assign tasks across machines
- Share code/context between sessions

## Broker

The message broker runs at https://agent-chat.awpdemon.com
